üìù Integration Notes

Important: Make sure to host the support-widget.html file on your server or CDN before embedding it on other websites.

Methods to Pass Dimensions:

  1. URL Parameters: Add ?width=400px&height=600px to the iframe src
  2. PostMessage API: Send resize messages from parent window to iframe
  3. Iframe Attributes: Set width and height directly on the iframe element

Recommended Minimum Dimensions:

  • Width: 320px minimum (for mobile compatibility)
  • Height: 400px minimum (to show content properly)
  • Optimal: 400px √ó 600px for desktop, 100% √ó 700px for responsive

Cross-Origin Communication:

The widget listens for messages from the parent window. You can send configuration updates like this:

const iframe = document.getElementById('support-widget'); // Wait for iframe to be ready window.addEventListener('message', (event) => { if (event.data.type === 'IFRAME_READY') { // Iframe is ready, you can now send messages iframe.contentWindow.postMessage({ type: 'RESIZE', width: '450px', height: '700px' }, '*'); } });
